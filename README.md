# Retrieval-Augmented-Generation-system
An AI-driven report automation system built with FastAPI, OpenAI GPT models, and ChromaDB that generates professional forensic engineering reports. It uses RAG for context retrieval, GPT-4.1 for report writing, GPT-4o-mini for caption enhancement, and integrates spaCy, Airtable, and PDF rendering for a complete end-to-end workflow.
